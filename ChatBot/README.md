# 🤖 LangChain AI Chatbot

A simple terminal-based AI chatbot built with **LangChain**, **NVIDIA NIM API**, and **Rich**.

## ✨ Features

* 💬 AI Chatbot
* ⚡ Streaming responses
* 📝 Markdown output
* 🧠 Conversation memory
* 🎨 Beautiful terminal interface

## 📦 Requirements

* Python 3.10+
* NVIDIA API Key

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/langchain-chatbot.git
cd langchain-chatbot
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## 🔑 Environment Variables

Create a `.env` file:

```env
NVIDIA_API_KEY=your_api_key_here
```

## ▶️ Run

```bash
python app.py
```
